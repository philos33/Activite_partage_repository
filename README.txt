Le programme est un algorithme de simulation du jeu de Monty-Hall avec visualisation des résultats.

Le projet comprend un notebook jupyter en python avec un module contenant une fonction appelée par le notebook.

Le jeu de Monty-Hall consiste à trouver un trésor caché derrière une porte, parmi 3 portes.
- Le joueur choisit une première porte. Une des deux autres portes est alors ouverte (une porte ne conduisant pas au trésor).
- Il reste alors deux portes et le joueur doit choisir s'il reste sur son premier choix ou s'il change de porte.

La simulation montre que s'il reste sur la premiere porte choisie, il a 33% de chance de gagner, s'il change de porte, il a alors 66% de chance de gagner.
